## 1. 概述

本文档详细描述了基于SAP CAP框架和PostgreSQL数据库的后端服务所暴露的OData V4 API。这些服务为ORBAI Core前端应用提供数据访问和业务逻辑处理能力。

- **基础URL**: `http://localhost:5885`

## 2. 差旅服务 (Travel Service)

该服务用于管理差旅相关的项目、行程和发票信息。

- **服务路径**: `/travel`
- **元数据URL**: `http://localhost:5885/travel/$metadata`
